# Configuration

## RabbitMQSourceConnector

Connector is used to read from a RabbitMQ Queue or Topic.

```properties
name=connector1
tasks.max=1
connector.class=com.github.jcustenborder.kafka.connect.rabbitmq.RabbitMQSourceConnector

# Set these required values
rabbitmq.queue=
kafka.topic=
```
